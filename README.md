> # 目录结构 
>> - bin：打包生成目录
>> - business：业务实现
>> - common：通用定义，如基础入参、出参，常量，配置对象等。
>> - resources：配置文件、模板文件
>> - timer：定时任务
>> - utils：工具包，如短信、邮件、钉钉、加解密、数据库、http工具等。
>> - web：http接口
>> - websocket：websocket服务
>> - main.go：程序入口
